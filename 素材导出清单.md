# Figma 素材导出清单（看展 + 个人介绍）

> 只需导出下面列出的「照片 / 插画 / 手机 mockup / logo」。
> 渐变、色块、气泡、大数字、装饰线条、软件图标、联系方式图标 **都不用导**，我用 CSS / 标准图标库还原。

---

## 一、通用导出设置

- **格式**
  - 照片、复杂插画 → **JPG**（质量 80–90，文件更小）
  - 需要透明背景的（logo、手机 mockup、贴纸/徽标）→ **PNG**
- **倍率**
  - 大图（背景照、插画、手机 mockup）→ **2x**
  - 小图（logo、徽标）→ **3x**
- **命名**：全部小写、用连字符 `-`，严格按下面给的文件名（这样我能直接对应到代码里）。
- **怎么导**：在 Figma 里选中对应元素 → 右侧 Export 面板 → 加一条导出设置（选格式和倍率）→ Export。

## 五、小提示

- 拿不准某张算不算"必须导"——**宁可多导**，我会按需取用，没用到的忽略即可。
- 手机 mockup 请**连同屏幕里的内容一起整台导出**（导成一张 PNG），不用拆开。
- 导出时如果某个元素带阴影/模糊，导出的 PNG 通常会包含；若发现丢了阴影，告诉我，我用 CSS 补。
- 全部导完、文件夹连给我后，我就开始搭 Astro 骨架并逐页还原。
